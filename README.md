# Chicken Geo

Un jeu pour apprendre la géographie en s'amusant ! Une poule vole vers un globe que tu peux faire tourner pour qu'elle atterrisse sur le bon pays.

## Comment jouer

1. Un nom de pays s'affiche à l'écran
2. Fais tourner le globe pour positionner le bon pays sous le viseur
3. La poule plonge et atterrit — est-ce le bon pays ?
4. Appuie sur **Accélérer** pour aller plus vite si tu es sûr(e) de toi !

### Contrôles

- **Glisser (souris / un doigt)** — faire tourner le globe
- **Deux doigts (rotation)** — incliner le globe (libre aux pôles)
- **Bouton Accélérer** — accélère la descente de la poule (×3)

## Lancer le projet

Ouvre `index.html` dans ton navigateur, ou lance un serveur local :

```bash
python3 -m http.server 8080
```

Puis ouvre `http://localhost:8080`.

## Technologies

- HTML, CSS, JavaScript (vanilla)
- [Three.js r128](https://threejs.org/) (CDN) pour le rendu 3D du globe
