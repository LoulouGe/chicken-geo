# Chicken Geo

Un jeu pour apprendre la géographie en s'amusant ! Une poule vole vers un globe que tu peux faire tourner pour qu'elle atterrisse sur le bon pays.

## Comment jouer

1. Un nom de pays s'affiche à l'écran
2. Fais tourner le globe pour positionner le bon pays sous la poule
3. La poule s'envole et atterrit — est-ce le bon pays ?

## Lancer le projet

Ouvre `index.html` dans ton navigateur, ou lance un serveur local :

```bash
python3 -m http.server 8080
```

Puis ouvre `http://localhost:8080`.

## Technologies

HTML, CSS, JavaScript (vanilla) — aucune dépendance externe.
